
# Домашнее задание: Тестирование готовых программ

Этот репозиторий содержит решение домашнего задания по теме **написание тестов к готовым программам**.  
Цель проекта — продемонстрировать умение анализировать существующий код, понимать его логику и писать проверки (тесты) для его корректной работы.

## Задача

В рамках задания был предоставлен фрагмент кода на языке Python, реализующий функцию `solve(receipts)`, которая:

- Принимает список чисел (`receipts`) — номера чеков.
- Возвращает каждый третий элемент начиная с **третьего** (индексация с 0: элементы с индексами 2, 5, 8, ...).
- Также возвращает количество таких элементов.


## Реализация

Функция использует срез `receipts[2::3]`, что означает:
- Начало с индекса 2,
- До конца списка,
- С шагом 3.

Тесты проверяют:
- Корректность возвращаемого списка.
- Правильность подсчёта количества элементов.
- Поведение на коротких списках (меньше 3 элементов).
